# UADER_IS2_Guiffrey

## Descripción del Repositorio
Repositorio de trabajos prácticos de la materia **Ingeniería de Software II** de la Universidad Autónoma de Entre Ríos.

## Estructura del Proyecto

### Carpetas
- **src**: Código fuente de los programas
- **doc**: Documentación del proyecto
- **bin**: Archivos binarios ejecutables
- **script**: Scripts auxiliares

### Tecnologías utilizadas
- Python 3
- Git
- GitHub
- Matplotlib

## Programas

### primos.py
Programa que calcula e imprime todos los números primos entre 1 y 500.

#### Pasos para ejecutar primos.py
1. Clonar el repositorio
2. Navegar a la carpeta src
3. Ejecutar con `py primos.py`

### factorial.py
Programa que calcula el factorial de un número o rango de números.

#### Modos de uso
1. `py factorial.py 10` - calcula el factorial de 10
2. `py factorial.py 4-8` - calcula factoriales del 4 al 8
3. `py factorial.py -10` - calcula factoriales del 1 al 10
4. `py factorial.py 5-` - calcula factoriales del 5 al 60
5. `py factorial.py` - solicita el número al usuario

### factorial_OOP.py
Versión orientada a objetos de factorial.py usando la clase **Factorial** con los métodos `__init__` y `run(min, max)`.

### collatz.py
Programa que calcula la conjetura de Collatz para los números entre 1 y 10000 y genera un gráfico con Matplotlib mostrando las iteraciones de cada número.

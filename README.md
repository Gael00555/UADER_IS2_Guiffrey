# UADER_IS2_Guiffrey

## Descripción del Repositorio
Repositorio de trabajos prácticos de la materia **Ingeniería de Software II** de la Universidad Autónoma de Entre Ríos.

## Estructura del Proyecto

### Carpetas
- **src**: Código fuente de los programas
- **doc**: Documentación del proyecto
- **bin**: Archivos binarios ejecutables
- **script**: Scripts auxiliares

### Tecnologías utilizadas
- Python 
- Git


## Programas

### primos.py
Programa que calcula e imprime todos los números primos entre 1 y 500.

#### Pasos para ejecutar primos.py
1. Clonar el repositorio
2. Navegar a la carpeta src
3. Ejecutar con `py primos.py`
